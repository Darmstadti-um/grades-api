# Быстрый старт

## 1. Клонирование и запуск

```bash
git clone <repository-url>
cd grades-api
make up
```

## 2. Проверка работы

Откройте в браузере: http://localhost:8000/docs

## 3. Загрузка данных

Используйте пример CSV файла из `EXAMPLE.md` или создайте свой по формату:
- Разделитель: `;`
- Заголовки: `Дата;Номер группы;ФИО;Оценка`
- Дата: `DD.MM.YYYY`
- Оценка: от 2 до 5

## 4. Тестирование

```bash
make itest
```

## Остановка

```bash
make down
```
